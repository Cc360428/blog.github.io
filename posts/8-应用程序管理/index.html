源码安装

  • 主要是依赖 make 和Makefile 编译
  • make install ，执行完之后，安装一般会被安装在/uer/local下对应的文件夹，部分二进制文件放在usr/bin或在/usr/local/bin 对应的头文件被安装至/usr/include，帮助文档安装至/usr/local/share ,如果指定目录请加到环境变量里面去

yum install

  • 下载地址【http://nginx.org/】
  • 创建预知准备好的Nginx目录
  • rpm、源码、yum install都可以
