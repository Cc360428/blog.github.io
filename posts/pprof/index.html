使用浏览器直接查看

  • 定位问题

    • 查看服务使用cpu
    • 内存使用情况磁盘和io使用情况

  • 使用工具

    • ps -aux grep qiuqiu ——–> 查找服务
    • stop -p pid ——–> CPU 内存
    • iotop -p pid ——–> 磁盘io

  • 分析问题

    • 生成pprof文件

      • go tool pprof http://172.13.5.17:14611/debug/pprof/profile （默认放在 cd $HOME/pprof/ ）
      • 可以看到会生成这样的文件 /Users/cc/pprof/pprof.qiuqiu.samples.cpu.002.pb.gz

    • go tool pprof -http=0.0.0.0:999 /Users/cc/pprof/pprof.qiuqiu.samples.cpu.002.pb.gz （使用浏览器预览生成的文件查看服务状态）
